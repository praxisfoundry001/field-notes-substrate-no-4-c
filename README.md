# Field Notes from the Substrate — Translations No. 4

## c and How Fast Reality Can Hand Off a Change

**A Structural Flow Translation of the Coherent Propagation Ceiling**

## What is this?

This is the fourth installment in *Field Notes from the Substrate*, a working-note series translating physics-facing concepts through Structural Flow.

This note translates **c** not first as “the speed of light,” but as the **coherent propagation ceiling**: the invariant ceiling under which carried consequence can be lawfully taken up through density-held spacetime as the same answerability-bearing carried burden.

## Why should I care?

Light reveals **c** cleanly. That familiar description is not wrong.

But this note asks what structural role **c** must be carrying if lawful propagation is possible at all.

The proposed answer is that **c** names the ceiling on lawful same-burden handoff: how fast reality can hand off a carried change without losing sameness, boundary, uptake, or coherent order.

## What is the claim?

The core claim is:

**c is the coherent propagation ceiling.**

More plainly:

**c is the shared ceiling on how fast carried consequence can become carried consequence again.**

In this note, “coherent propagation ceiling” names the whole-order role. “Coherent uptake ceiling” names the local handoff face of that same role.

The formal translation term is **coherent propagation ceiling**.

It does not derive Special Relativity.

It does not derive Maxwell’s equations.

It does not derive photon ontology.

It does not introduce a preferred frame.

It does not solve the black-hole information paradox.

It does not claim that every appearance of **c** in physics has been translated here.

The note translates one structural face of **c**: the propagation-ceiling face.

## How was this done?

The note develops three linked burdens:

1. A **distinguishable carried update** is a retained, bounded, consequence-bearing difference that remains available for lawful continuation and comparison as the same update across local structural passage.

2. Same-update handoff requires the same carried burden to remain recognizable while a new carrier actually takes it up.

3. Two working lemmas support the conditional role-shape translation:

**Working Lemma 1:** bounded same-register answerability is finite.

**Working Lemma 2:** zero-duration uptake is not uptake.

Together, these yield a finite throughput burden at role-shape scope. Lawful same-burden propagation at this register needs a ceiling. Recognition makes that ceiling transportable across lawful local variation. Coherence makes it one jointly holdable law.

## Failure signature

The candidate failure signature is:

**lawful local uptake burdens exist, but their ceiling cannot be held as one invariant propagation order across lawful local descriptions.**

Compressed:

**finite uptake fails to cohere.**

The primary failure regime is **Drift**.
